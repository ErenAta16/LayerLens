# LayerLens Experiments

This directory contains experimental scripts for testing and validating LayerLens functionality.

## Available Experiments

### `fine_tuning_experiment.py`
Experimental fine-tuning workflow comparing LayerLens configurations with baselines.

### `test_configurations.py`
Tests LayerLens with various configuration combinations to validate robustness.

## Running Experiments

```bash
# From project root
python experiments/fine_tuning_experiment.py
python experiments/test_configurations.py
```

## Notes

These scripts are for development and testing purposes. For production usage, refer to the demos in `demos/` directory.
